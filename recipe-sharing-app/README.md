## Setup & Run Locally

Clone the repo:

```bash
git clone https://github.com/Mia06-coder/alx-fe-reactjs.git
cd alx-fe-reactjs/recipe-sharing-app
```

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open your browser at [http://localhost:5173](http://localhost:5173).
